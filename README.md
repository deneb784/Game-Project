## 게임 기획

## 게임 규칙 및 설명

해당 게임은 기존 보드게임 Dixit과 동일하게 진행되도록 설계하였다.

### 게임 규칙
- 플레이어는 사용자와 AI 4명으로 구성된다.
- 카드는 총 108장으로 구성되어 있다.

### 게임 진행

1. 게임이 시작되면 모든 플레이어는 랜덤한 각각의 카드 5장씩을 받는다.
2. 각각의 플레이어는 돌아가면서 이야기꾼이 된다.
3. 이야기꾼은 자신의 카드 중 한 장을 선택하여, 해당 카드에 대해 설명한 후 다른 플레이어에게 보이지 않게 제출한다.
4. 이야기꾼이 아닌 플레이어는 해당 설명을 듣고, 자신이 가진 카드 중 해당 설명과 가장 유사한 카드를 보이지 않게 제출한다.
5. 모든 플레이어가 제출을 완료하면, 제출한 카드들을 모두 공개한다.
6. 이야기꾼이 아닌 플레이어들은 해당 카드들을 보고, 이야기꾼이 냈을것으로 생각되는 카드에 투표한다.
7. 투표가 끝나면, 투표 결과에 따라 점수를 획득한다.
   - 이야기꾼의 카드를 모든 플레이어가 맞추거나, 또는 어떤 플레이어도 맞추지 못했다면 이야기꾼을 제외한 전원이 2점을 획득한다.
   - 그렇지 않다면 이야기꾼과 정답을 맞춘 플레이어가 3점을 획득한다.
   - 이야기꾼이 아닌 플레이어가 자신이 제출한 카드에 투표를 받았다면, 투표를 받은 수만큼 1점씩 획득한다.
8. 점수 계산이 끝나면, 모든 플레이어가 카드를 1장씩 보충한다.
9. 2~8번의 과정을 30점이 넘는 플레이어가 나올때까지 진행한다.
